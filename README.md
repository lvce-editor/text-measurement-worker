# Text Measurement Worker

> A Web Worker for the completion functionality in Lvce Editor.

## Contributing

```sh
git clone git@github.com:lvce-editor/editor-commands-worker.git &&
cd editor-commands-worker &&
npm ci &&
npm test
```
